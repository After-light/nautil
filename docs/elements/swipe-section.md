# SwipeSection

```js
import { SwipeSection } from 'nautil'

<SwipeSection>
  ...
</SwipeSection>
```

## props

- sensitivity: Number, distance from screen border
- distance: Number, moved distance before onStart
- disabled: Boolean,
- direction: enumerate(['left', 'right', 'both']),
- throttle: Number,
- onStart
- onMove
- onEnd
- onCancel
- onReach
- onUnreach
